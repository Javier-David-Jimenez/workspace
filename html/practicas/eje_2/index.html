Etiqueta br

El elemento HTML line break br produce un salto de línea en el texto (retorno de carro).
Es útil para escribir un poema o una dirección, donde la división de las líneas es significante.

Etiqueta HR

Se trata de la línea horizontal (horizontal rule) cuya marca, única, es HR.

Mediante la inserción de esta etiqueta entre dos párrafos, éstos quedan separados por una línea horizontal con aspecto de relieve.

Etiqueta strong

Definición. El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto

Etiqueta a y atributo href

La a designa un texto de anclaje. Esta etiqueta HTML define el punto inicial y final del hipervínculo.
El atributo href se utiliza dentro del texto de anclaje y define el link de destino objetivo.
Además de una página web interna o externa, aquí también se pueden enlazar, por ejemplo, documentos PDF o imágenes Ir a Google

atributo target

Target="_blank", coloquialmente llamado target_blank, es un atributo HTML que se puede colocar en un
enlace para indicar que la página de destino hacia la que apunta se abra en otra ventana del navegador.
El atributo target indica al navegador dónde abrir la página o documento enlazado. Ir a Google
